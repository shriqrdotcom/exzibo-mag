### STEP 6 — Check for Build Errors

If the Vercel build fails, the most common reasons are:

1. Missing environment variables in Vercel (Supabase keys not added)
2. A package that works in Replit but isn't in package.json 
   (run: npm install → check if package.json was updated)
3. TypeScript or lint errors that Replit ignored but Vercel's build catches

Run this in Replit Shell to simulate what Vercel will build:
  npm run build

If it fails locally in Replit too, fix those errors first before pushing.
